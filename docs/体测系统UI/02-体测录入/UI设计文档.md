# 体测录入 UI设计文档

## 1. 页面定位

- 菜单：体测管理 / 体测录入
- 目标：按体测批次录入学生各单项成绩并自动算分
- 页面类型：批次选择 + 学生选择 + 单项录入表

## 2. 路由与权限

- 路由：`/fitness/entry`
- 组件：`views/Vadmin/Fitness/Entry/Entry`
- 权限码：`fitness.score.entry`

## 3. 页面结构

- 筛选区：批次、学校、年级、班级、学生（关键词搜索）
- 信息区：当前标准版本、地区、年份、学段、性别
- 录入区：按单项录入成绩，自动回显分值与等级
- 结果区：保存结果、批量导入结果、异常提示

## 4. 规则

- 创建体测批次必须绑定标准版本。
- 允许仅录入成绩，由引擎自动计算分值。
- 冲突规则默认“就低不就高”。
- 导入识别规则需人工确认后生效。
