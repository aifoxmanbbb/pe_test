# 成绩总览 UI设计文档

## 1. 页面定位

- 菜单：体考管理 / 成绩总览
- 目标：按批次、学校、年级、班级查看体育项目成绩、分值与阈值对比
- 页面类型：筛选 + KPI + 图表 + 学校分组班级列表

## 2. 路由与权限

- 路由：`/pe/overview`
- 组件：`views/Vadmin/PE/Overview/Overview`
- 权限码：`pe.analysis.overview`
- 数据范围：
  - 管理员：全局
  - 学校主管：本校
  - 学校老师/教练：本班

## 3. 页面结构

- 顶部 Tab：`初中 | 高中`
- 筛选区：`批次 -> 学校 -> 年级 -> 班级`
- 主体 Card（查询按钮下方）：
  - 主标题：`某批次-某学校/所有学校（管理视角可不筛选学校） 体育项目情况分析`
  - KPI：参考人数、平均分、及格率、优秀率、满分率
  - 图表：
    - 项目均分柱状图（含门槛项和其他项目）
    - 班级分层率图（及格率/优秀率/满分率）
    - 批次趋势图
  - 列表：学校分组班级成绩列表（学校列合并单元格）

## 4. 展示规则

- 全局用词统一：`学生`、`成绩`、`及格/优秀/满分`。
- 颜色统一：
  - 及格/及格率：黄色
  - 不及格：红色
  - 优秀/优秀率：绿色
  - 满分/满分率：纯黑色
- 所有图表与列表必须可见：`成绩`、`分值`、`及格阈值`、`优秀阈值`、`满分阈值`。
- 门槛项同样展示成绩与分值。
- 不统计“良好率”。

## 5. 交互规则

- 移除筛选项：项目、性别、学生关键词、时间范围。
- 管理视角允许学校留空（查看所有学校）。
- 柱状图顶部显示成绩或分值标签。
- 列表去掉外框，按学校分组展示。
- Card 主标题样式：居中 + 加粗 + 大字号（高于所有子标题）。

## 6. 接口清单

- `GET /vadmin/pe/analysis/overview/kpi`
- `GET /vadmin/pe/analysis/overview/item-avg`
- `GET /vadmin/pe/analysis/overview/class-rate`
- `GET /vadmin/pe/analysis/overview/trend`
- `GET /vadmin/pe/analysis/overview/class-list`

## 8. 状态处理

- 无权限数据：提示“当前角色无可见数据”。
- 空数据：保留筛选区 + 空态提示。
- 接口异常：提示错误并保留最后一次成功结果。
