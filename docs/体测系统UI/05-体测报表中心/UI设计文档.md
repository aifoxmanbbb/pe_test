# 体测报表中心 UI设计文档

## 1. 页面定位

- 菜单：体测管理 / 体测报表中心
- 目标：导出学生/班级/年级体测报表
- 页面类型：参数配置 + 同步导出 + 历史记录

## 2. 路由与权限

- 路由：`/fitness/report`
- 组件：`views/Vadmin/Fitness/Report/Report`
- 权限码：`fitness.report.export`

## 3. 结构

- 参数区：报表类型、批次、学校、年级、班级、学生关键词
- 字段区：学生信息、项目成绩、项目评分成绩、单项及格/优秀/满分状态、标准版本号
- 结果区：成功/失败 + 下载
- 历史区：最近导出记录

## 4. 口径规则

- 不导出综合分、总分、总平均分。
- 不导出总体及格/优秀/满分状态。
- 仅导出单项口径字段。
