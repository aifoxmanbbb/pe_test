# 体测标准中心 UI设计文档

## 1. 页面定位

- 菜单：体测管理 / 体测标准中心
- 目标：管理体测评分标准版本（地区/年份/学段/性别）
- 页面类型：标准列表 + 导入校对 + 详情弹窗

## 2. 路由与权限

- 路由：`/fitness/standard`
- 组件：`views/Vadmin/Fitness/Standard/Standard`
- 权限码：`fitness.standard.list`、`fitness.standard.import`、`fitness.standard.confirm`、`fitness.standard.publish`、`fitness.standard.update`

## 3. 规则

- 支持 PDF 识别导入、手工录入、表格导入。
- 识别结果必须人工确认。
- 支持复制历史标准生成新版本。
- 标准发布后允许修改。
- 分值段规则与阈值规则统一纳入详情弹窗查看。
