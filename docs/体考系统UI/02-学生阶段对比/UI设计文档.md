# 学生阶段对比 UI设计文档

## 1. 页面定位

- 菜单：体考管理 / 学生阶段对比
- 目标：单学生多批次成绩、分值、项目变化分析
- 页面类型：筛选 + 学生档案 + 指标卡 + 趋势图 + 批次列表

## 2. 路由与权限

- 路由：`/pe/analysis/student`
- 组件：`views/Vadmin/PE/Analysis/Student/Student`
- 权限码：`pe.analysis.student`
- 学生自助：`pe.analysis.student.self`（仅本人）

## 3. 页面结构

- 顶部 Tab：`初中 | 高中`
- 筛选区：学校、年级、班级、学生（支持姓名/学号/联系方式关键词搜索）
- 主体 Card（查询按钮下方）：
  - 主标题：`某学校-学号-学生 体育项目情况分析`
  - 学生档案：姓名、性别、联系方式、学校、入学年、年级、班级、学号、学段
  - 指标卡：最新总分、历史最高总分、及格项目数、不及格项目数、优秀项目数、满分项目数、优秀项目、满分项目
  - 图表：总分变化趋势、各项目变化柱状图
  - 列表：批次成绩列表（单学生）

## 4. 展示规则

- 页面仅允许单个学生展示。
- 批次列表每行一个批次，每项目使用单单元格上下展示：上行`成绩`，下行`分值`。
- 门槛项也展示成绩与分值；门槛项不及格仍允许后续项目录入，且不做额外拦截标记。
- 列表末尾字段：老师评语。
- 颜色统一：
  - 及格项目数：黄色
  - 不及格项目数：红色
  - 优秀：绿色
  - 满分：纯黑色

## 5. 交互规则

- 移除筛选项：项目、学生关键词独立字段、时间范围。
- 按钮文案：`导出图表`。
- 图表标题短化，Card 主标题居中加粗且字号高于子标题。
- 列表、学生档案区去外框（使用主 Card 包裹）。

## 6. 接口清单

- `GET /vadmin/pe/students/options`
- `GET /vadmin/pe/analysis/student/profile`
- `GET /vadmin/pe/analysis/student/summary`
- `GET /vadmin/pe/analysis/student/total-trend`
- `GET /vadmin/pe/analysis/student/item-trend`
- `GET /vadmin/pe/analysis/student/detail-list`

## 8. 状态处理

- 未选择学生：仅展示筛选区与提示。
- 无历史批次：显示档案与空态。
- 成绩解析异常：行内提示“待人工确认”。
