# 报表中心 UI设计文档

## 1. 页面定位

- 菜单：体考管理 / 报表中心
- 目标：按学生/班级/年级维度同步导出体考报表
- 页面类型：参数配置 + 同步导出结果 + 导出历史

## 2. 路由与权限

- 路由：`/pe/report`
- 组件：`views/Vadmin/PE/Report/Report`
- 权限码：`pe.report.export`
- 学生自助：`pe.report.student.self.export`（仅本人）

## 3. 页面结构

- 顶部 Tab：`初中 | 高中`
- 参数区：报表类型、批次、学校、年级、班级、学生关键词、导出字段
- 结果区：导出成功/失败反馈
- 历史区：最近导出记录

## 4. 字段口径

- 全局用词统一“学生”“成绩”。
- 默认导出字段：
  - 学生基础信息
  - 项目成绩、项目评分成绩、总分
  - 及格/优秀/满分阈值
  - 及格/优秀/满分状态
  - 标准版本号

## 5. 交互规则

- 移除“时间范围”筛选。
- 点击“生成报表”后同步返回下载入口（不做异步导出）。
- 失败时展示错误信息并支持重试。

## 6. 接口清单

- `POST /vadmin/pe/reports/student/export`
- `POST /vadmin/pe/reports/class/export`
- `POST /vadmin/pe/reports/grade/export`

## 8. 状态处理

- 参数不完整：阻断提交并定位字段。
- 权限不足：导出按钮禁用并提示。
- 数据量过大：提示缩小筛选范围。
