# 班级体测分析 UI设计文档

## 1. 页面定位

- 菜单：体测管理 / 班级体测分析
- 目标：单班级在历史批次下的单项变化与班内学生单项表现分析
- 页面类型：单项统计卡 + 单项趋势图 + 全量排名列表

## 2. 路由与权限

- 路由：`/fitness/analysis/class`
- 组件：`views/Vadmin/Fitness/Analysis/Class/Class`
- 权限码：`fitness.analysis.class`

## 3. 页面结构

- 筛选区：批次、学校、年级、班级
- 主体 Card：`某批次-某学校-年级-班级 体质测试情况分析`
- 统计卡：学生人数、项目数、单项记录数、不及格单项记录、满分单项记录
- 图表：
  - 批次单项均分趋势
  - 当前批次单项达成率
- 列表：班级学生全量明细（各项目成绩/分值）

## 4. 口径规则

- 不展示综合分、总分、总平均分。
- 不展示总体及格率/优秀率/满分率与总体状态。
- 仅展示单项成绩、单项分值、单项达成情况。
