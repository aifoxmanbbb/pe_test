# 学生体测分析 UI设计文档

## 1. 页面定位

- 菜单：体测管理 / 统计分析 / 学生体测分析
- 目标：单个学生多批次单项成绩、单项分值与单项状态变化分析
- 页面类型：学生档案 + 单项统计卡 + 单项趋势图 + 批次明细

## 2. 路由与权限

- 路由：`/fitness/analysis/student`
- 组件：`views/Vadmin/Fitness/Analysis/Student/Student`
- 权限码：`fitness.analysis.student`

## 3. 页面结构

- 顶部 Tab：`初中 | 高中`
- 筛选区：学校、年级、班级、学生（支持关键词搜索）
- 主体 Card：`某学校-学号-学生 体质测试情况分析`
- 学生档案：姓名、性别、联系方式、学校、入学年、年级、班级、学号、学段
- 统计卡：已测项目数、及格项目数、不及格项目数、优秀项目数、满分项目数、不及格项目、优秀/满分项目
- 图表：
  - 单项分值变化趋势
  - 单项状态数量变化
- 明细：批次单项成绩列表（每项成绩/分值）

## 4. 口径规则

- 不展示综合分、总分、总平均分。
- 不展示总体及格/优秀/满分状态。
- 仅展示单项口径（单项成绩、分值、状态）。
