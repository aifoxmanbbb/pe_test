# 班级对比分析 UI设计文档

## 1. 页面定位

- 菜单：体考管理 / 班级对比分析
- 目标：单个班级在历史批次下的项目均分趋势与班内学生排名分析
- 页面类型：筛选 + KPI + 历史趋势图 + 全量排名表

## 2. 路由与权限

- 路由：`/pe/analysis/class`
- 组件：`views/Vadmin/PE/Analysis/Class/Class`
- 权限码：`pe.analysis.class`
- 数据范围：管理员全局 / 学校主管本校 / 学校老师教练本班

## 3. 页面结构

- 顶部 Tab：`初中 | 高中`
- 筛选区：批次、学校、年级、班级
- 主体 Card（查询按钮下方）：
  - 主标题：`某批次-某学校-年级-班级 体育项目情况分析`
  - KPI：班级平均分、及格率、优秀率、满分率
  - 图表：
    - 历史批次平均分趋势（4条线）
    - 历史批次分项目趋势（柱状）
  - 底部列表：班级学生全量排名表

## 4. 展示规则

- 页面仅支持单个班级展示。
- 排名表必须全量展示，不做分页截断。
- 每行一个学生，各项目（含门槛项）单元格上下展示`成绩/分值`。
- 列表末尾字段：老师评语。
- 及格黄色，不及格红色，优秀绿色，满分黑色。

## 5. 交互规则

- 移除筛选项：性别、学生关键词、时间范围。
- 图表标题使用短名称，主 Card 标题居中加粗且大于子标题。
- 排名列表放页面底部，去外框。

## 6. 接口清单

- `GET /vadmin/pe/analysis/class/kpi`
- `GET /vadmin/pe/analysis/class/history-avg-trend`
- `GET /vadmin/pe/analysis/class/history-item-trend`
- `GET /vadmin/pe/analysis/class/rank-full`

## 8. 状态处理

- 无班级权限：提示“当前账号未绑定可见班级”。
- 无成绩数据：保留筛选与表头，显示空态。
