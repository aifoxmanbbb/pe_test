# 年级体测分析 UI设计文档

## 1. 页面定位

- 菜单：体测管理 / 年级体测分析
- 目标：单年级各班单项均值、单项达成率、单项历史趋势分析
- 页面类型：单项统计卡 + 多图表 + 班级统计表

## 2. 路由与权限

- 路由：`/fitness/analysis/grade`
- 组件：`views/Vadmin/Fitness/Analysis/Grade/Grade`
- 权限码：`fitness.analysis.grade`

## 3. 页面结构

- 筛选区：批次、学校、年级
- 主体 Card：`某批次-某学校-年级 体质测试情况分析`
- 统计卡：班级数、学生人数、单项记录数、不及格单项记录、满分单项记录
- 图表：
  - 各班单项均分对比
  - 各班单项达成率对比（按选定项目）
  - 各班批次单项均分趋势（按选定项目）
- 列表：各班单项统计（单项平均成绩/平均分 + 单项达成情况）

## 4. 口径规则

- 不提供综合分、总分、总平均分。
- 不提供总体及格率/优秀率/满分率与总体状态。
- 仅提供单项维度统计与对比。
