# 评分标准中心 UI设计文档

## 1. 页面定位

- 菜单：体考管理 / 评分标准中心
- 目标：管理地区/年份/学段/性别维度的标准版本与分值规则
- 页面类型：标准列表 + 详情弹窗 + 导入校对

## 2. 路由与权限

- 路由：`/pe/standard`
- 组件：`views/Vadmin/PE/Standard/Standard`
- 权限码：`pe.standard.list`、`pe.standard.create`、`pe.standard.import`、`pe.standard.confirm`、`pe.standard.publish`、`pe.standard.update`

## 3. 页面结构

- 筛选区：地区、年份、学段、状态、关键词
- 列表区：标准版本列表（版本号、来源、状态、引用批次）
- 详情区：通过“详情”弹窗展示，不在页面底部常驻
- 操作区：导入 PDF、导入表格、手工录入、人工确认、发布、复制新版

## 4. 关键规则

- 创建测试批次时必须手工选择标准版本。
- 同名项目在不同地区/年份/性别/学段可配置不同阈值与分值规则。
- 红头文件识别结果必须人工确认后可发布。
- 历史标准可复制生成新版标准，必须有标准版本号。
- 标准发布后允许直接修改。
- 规则冲突固定策略默认“就低不就高”。

## 5. 阈值与分值展示

- 标准详情弹窗内展示项目规则表：
  - 项目
  - 性别
  - 分值段（成绩区间 -> 分值）
  - 满分阈值
  - 优秀阈值
  - 及格阈值
  - 计分模式

## 6. 接口清单

- `GET /vadmin/pe/standards/list`
- `POST /vadmin/pe/standards/create`
- `POST /vadmin/pe/standards/import`
- `POST /vadmin/pe/standards/parse/{task_id}/confirm`
- `POST /vadmin/pe/standards/{id}/publish`
- `POST /vadmin/pe/standards/{id}/clone`
- `PUT /vadmin/pe/standards/{id}`
- `GET /vadmin/pe/standards/{id}/detail`

## 8. 状态处理

- 识别失败：显示原因并可下载原始文件。
- 冲突未解决：阻断发布并定位冲突项。
- 草稿未发布：批次创建不可选。
