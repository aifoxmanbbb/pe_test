# 体测总览 UI设计文档

## 1. 页面定位

- 菜单：体测管理 / 体测总览
- 目标：按批次、学校、年级、班级查看体质测试单项情况
- 页面类型：筛选 + 单项统计卡 + 单项图表 + 班级列表

## 2. 路由与权限

- 路由：`/fitness/overview`
- 组件：`views/Vadmin/Fitness/Overview/Overview`
- 权限码：`fitness.analysis.overview`

## 3. 页面结构

- 筛选区：批次、学校、年级、班级
- 主体 Card：`某批次-某学校/所有学校 体质测试情况分析`
- 统计卡：参考人数、项目数、单项记录数、不及格单项记录、满分单项记录
- 图表：
  - 单项平均分对比
  - 单项达成率对比（及格/优秀/满分）
  - 重点单项批次趋势
- 列表：学校分组班级体测统计（单项均成绩/均分 + 单项达成情况）

## 4. 口径规则

- 体测不提供综合分、总分、总平均分口径。
- 不提供总体及格率/优秀率/满分率与总体状态。
- 仅展示单项成绩、单项分值、单项及格/优秀/满分情况。
- 支持性别维度、年级阶段维度。
